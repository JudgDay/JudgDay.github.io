MySQL 8.0.16安装教程(windows 64位)

  1. 官网下载下载 MySQL下载地址
  2. 下载Windows 64位正式版

  3. 点击No Thanks,just start my download 便可以下载

  4. 下载完成后解压，解压后的目录

  5. 解压后的目录中没有my.ini文件，在安装根目录下自行创建my.ini（新建文本文件，将文件类型改为的.ini），写入基本配置：
  6. 配置文件中的路径要和实际存放的路径一致

  7. 在安装时，为避免出错使用管理员身份运行CMD，否则在安装时会报错，会导致安装失败的情况（进入C:\Windows\System32，右键cmd.exe以管理员身份运行）

  8. 打开CMD后进入mysql的bin目录

  9. 在MySQL目录下的bin目录下执行命令：mysqld --initialize --console

  10. 执行mysqld --install 命令，安装mysql服务

    执行sc delete mysql命令，删除MySQL，再执行mysqld --install安装服务命令

  11. 服务安装成功之后，执行net start mysql命令启动MySQL服务

  12. 执行mysql -u root -p命令登录mysql，密码为之前的临时密码

  13. 执行ALTER USER 'root'@'localhost' IDENTIFIED BY 'new password';命令修改mysql密码，new pasword为修改后的密码
